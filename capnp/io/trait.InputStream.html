A producer of bytes.

Required Methods

fn try_read(&mut self, buf: &mut [u8], min_bytes: usize) -> Result<usize>

Reads at least min_bytes into buf unless EOF is encountered first. Returns the number of bytes read.

Provided Methods

fn read(&mut self, buf: &mut [u8], min_bytes: usize) -> Result<usize>

Reads at least min_bytes into buf, returning the number of bytes read. If EOF is encountered first, returns an error.

fn read_exact(&mut self, buf: &mut [u8]) -> Result<()>

Reads into buf until it is full. Returns an error if EOF is encountered first.
